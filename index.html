Simple Web Projects

Programming the Web

Simple examples utilizing HTML, CSS, Javascript and other technologies to progressively explore the nuances associated with basic Web programming.

Word Frequency Counter

A primitive text area and input button facilitate an understanding of HashMaps utilizing Javascript Objects and sorting on both primary and secondary keys.

Weather Forecast Page

Geo-location data is fetched with Javascript to showcase local weather results. The page is dynamically modified dependent of the local weather. Users may share the forecast to social media.

Model View Controller

Simplified Model View Controller example to select a pizza size. This example does not facilitate the usage of a database. For a walkthrough Click Here.

API Viewer

Utilizing the API, content is generated dependent of the user search criterion and dynamically concatenated to the page. Fuzzy search is intended as a feature in the future.

Simple Blog

State is maintained utilizing a single session acquired upon login and released upon logout. Posts will expire after logout.

Voting Web Application

A database is used to track votes cast by registered users. Users may register with a social media account.

Social Media Clone

Users register/de-register an account that maintains state across multiple sessions (login/logout attempts). Users may create public posts visible to users within their network.
